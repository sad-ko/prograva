> Descripcion: Caso en el que Hercules jamas puede destruir todas las cabezas de la hidra.

```
in:

```

> Descripcion: En caso de tener que gastar todas sus espadas para vencer a la hidra.

```
in:

2 25 2

out:
5
0 3
```
